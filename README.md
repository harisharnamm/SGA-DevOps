# DevOps Assignment - Git & Docker

## Overview
This repository demonstrates foundational DevOps practices including Version Control using Git/GitHub and Containerization using Docker.

## Project Structure
- `index.html` - Home page
- `about.html` - About page with project details
- `contact.html` - Contact information page
- `Dockerfile` - Docker configuration for containerization
- `app.py` - Python application for Docker demonstration

## Git Commands Demonstrated
1. **git init** - Initialize a new Git repository
2. **git status** - Check the status of files in the repository
3. **git add** - Add files to staging area
4. **git commit** - Commit staged changes with a message
5. **git log** - View commit history
6. **git branch** - Create and list branches
7. **git switch** - Switch between branches

## Technologies Used
- HTML5 & CSS3
- Git & GitHub
- Docker & Docker Hub
- Python 3
